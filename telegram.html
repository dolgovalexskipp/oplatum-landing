Пример интеграции

Платёжная ссылка за пять строк кода

Python · aiogram
from oplatum import Oplatum

client = Oplatum(api_key="sk_live_...")

link = client.payment_links.create(
    amount=1500,
    currency="RUB",
    description="Подписка на канал",
    customer_id="tg_" + str(message.from_user.id)
)

# отправьте клиенту в Telegram
await message.answer(f"Оплата: {link.url}")
Node.js · grammy
import { Oplatum } from "@oplatum/sdk";

const client = new Oplatum({ apiKey: "sk_live_..." });

bot.command("buy", async (ctx) => {
  const link = await client.paymentLinks.create({
    amount: 1500,
    currency: "RUB",
    description: "Подписка",
    customerId: `tg_${ctx.from.id}`
  });
  await ctx.reply(`Оплата: ${link.url}`);
});
Возможности

Всё, что нужно боту с платежами

Платёжные ссылки

Сгенерируйте ссылку одним API-вызовом. Клиент открывает в Telegram, платит привычным способом — СБП, карта или USDT.

Рекурренты

Подписочные модели, автоматические списания, управление рекурренцией из SDK. Без костылей и саморучных таймеров.

Mini App checkout

Встраиваемый чекаут внутри Mini App. Один компонент, один API-ключ — клиент платит не выходя из вашего приложения.

Вебхуки и идемпотентность

Надёжные уведомления о статусах платежей, защита от дублей, безопасные повторы при сбоях сети.

Возвраты и споры

Refund и chargeback обрабатываются через API. Бесплатно, с полной отчётностью.

Song-sandbox и тест

Тестовая среда в день регистрации. Эмулятор всех методов оплаты, fake-клиенты, webhook replay.
