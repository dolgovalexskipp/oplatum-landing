07 · Для разработчиков

API уровня Stripe + MCP-сервер для AI-агентов

REST + webhooks с подписью HMAC-SHA256, идемпотентность, sandbox с тестовыми ключами. SDK для Node, Python, PHP. MCP-сервер — чтобы AI-агент сам интегрировал платежи в код.

Что есть в API
  • HMAC-SHA256 подпись запросов
  • Idempotency-Key на POST
  • Webhooks с retry policy
  • Sandbox: ak_test_ / sk_test_
  • Customers API + Events API
  • Checkout Sessions (hosted)
  • Возвраты через POST /refund
  • SDK Node / Python / PHP
  • Postman + интерактивная консоль
